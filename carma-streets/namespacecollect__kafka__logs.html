Python script to download and store kafka logs Opens the kafka docker container, then uses the example script kafka-console-consumer.sh to grab messages.

Creates a local folder with given name For each specified topic (default is all): downloads all messages between the given times (default is all) stores these messages to a file named .log Then once all topics are downloaded, zips up the folder
