Detailed Description

Introsort algorithm. Modified quicksort algorithm, that attempts to avoid the degenerate case when supplied a nearly-sorted input. The modification is that when this degenerate case is detected, heap-sort and insertion sort take over to avoid this worst case. Operates in the worst case O(nlgn).

This class also provides search methods, and number manipulators.
