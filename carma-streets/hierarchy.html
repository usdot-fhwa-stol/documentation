 Cstreets_service::streets_clock_singletonStreets Clock Singleton is a singleton that manages a CarmaClock object. The CarmaClock object is created by the CARMA Time Library (https://github.com/usdot-fhwa-stol/carma-time-lib) and is a wrapper object that allows for time based operation using chronos system time calls or an external source for time. Its constructor takes a boolean parameter, what when set to true relies on update calls to save and store the current time. When passed false, the CarmaClock object functions exactly as chrono system time calls including now() and sleep() calls, using the system time
