void NS3Client::process ( const std::shared_ptr< const std::vector< uint8_t > > &  data)
private

maintains the process thread

This will parse through the incoming UDP packets to find what looks like a valid J2735 message. The UPER scheme makes it hard to definitively know if a message is valid or just noise, so all possible messages from a packet are sent. This will create some false positives, but will ensure that a falsely identified message doesn't keep a real one from getting through.
