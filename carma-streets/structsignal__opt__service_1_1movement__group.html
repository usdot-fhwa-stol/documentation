Detailed Description

Movement group consists of up to two signal group ids which can concurrently be given green in a NEMA 8 Phase Dual Ring Traffic Signal Controller setup. Each single group will be from a different ring but both are inside the same barrier. A signal group pair with an entry of 0 indicates the movement group only consists of a single signal group (the non-zero value entry).
